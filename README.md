https://wokwi.com/projects/455725785722081281# Dilshaan-Bal(AIRGUARD PRO IOT)
AirGuard Pro is a closed-loop IoT air quality monitoring system built on ESP32 that continuously measures PM2.5 and gas levels using a Sharp dust sensor and MQ-135. It automatically triggers an exhaust fan within 2 seconds when pollutant thresholds are breached, and streams real-time AQI data to a ThingSpeak cloud dashboard. All readings are also displayed locally on an OLED screen for instant on-site feedback.Note: In the Wokwi simulation, potentiometers are used to simulate the MQ-135 gas sensor 
and Sharp PM2.5 dust sensor inputs as the actual components are unavailable in the 
simulator. The logic, thresholds, AQI calculations, and autonomous control behaviour 
remain fully functional and representative of the physical implementation.

Built as a practical demonstration of how low-cost IoT devices can replicate the core functionality of professional weather stations. integration remain fully functional and representative of the physical implementation.

https://www.tinkercad.com/things/86aiiVxFBZy-grand-stantia Dilshaan-Bal(Obstacle Avoidance Robot Using Ultrasonic Sensor)
An autonomous robot built on ESP32 that uses an ultrasonic sensor to detect obstacles in its path and automatically steers away without any human input. When an object is detected within a set distance threshold, the motor driver adjusts wheel direction in real-time to navigate around it, mimicking basic autonomous vehicle behaviour. Built as a hands-on prototype demonstrating closed-loop sensor-actuator integration and real-time decision making in embedded systems.
Note: The obstacle avoidance logic and ultrasonic sensor behaviour are fully 
simulated in Wokwi, accurately representing the real-world motor response 
and distance threshold detection of the physical robot.
